🧮 CALCULATION LOGIC (STRICT ORDER)
Total Project Value
→ Company Reserve (10–40%)
→ BD Bonus (from total)
→ Work Pool
→ Distributed by effort weights
→ Sub-employee deduction (from member share)
Formula:
Individual Share = (Weight / Total Weight) × Work Pool
⚠️ Rules:
If total weight = 0 → show error
All calculations update live
