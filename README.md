## 💰 Finance Engine

The core feature - complete project finance calculation system:

```
Total Value
  ↓
- Company Reserve (10-40%)
- BD Bonus (separate)
  ↓
= Work Pool
  ↓
÷ Total Weight (effort levels)
  ↓
× Individual Weight
  ↓
- Sub-employee Deductions (max 30%)
  ↓
= Final Payout
```

**Features:**
- Live calculations
- Donut chart (payouts)
- Waterfall chart (money flow)
- WhatsApp summary
- Approval workflow
- Project locking
